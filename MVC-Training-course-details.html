Live Instructor Led real time oriented training workshop on .NET Core. Those who have basic knowledge on C# and HTML can opt this training to gain real time oriented knowledge.

In this workshop, you will learn about .NET Core, middleware, MVC pattern, validations, helper controls and querying database using Entity Framework Core, Web API.

Training Objectives:

  • Understand .NET Core
  • Understand .NET Core Solution Structure
  • Understand ASP.NET Core middleware
  • Configure ASP.NET Core MVC
  • Use Dependency Injection in ASP.NET Core
  • Work with Entity Framework Core
  • Create REST Service using Web API
  • Publish theirs Apps on IIS

Pre-requisites
Basic knowledge on C# and HTML is mandatory

Tools/SDK/IDE
Visual Studio 2022 or higher, SQL Server 2019 or higher
